# 快速上手

### 安装

```bash
# 通过 npm
npm i vue3-hash-calendar

# 通过 yarn
yarn add vue3-hash-calendar

# 通过 pnpm
pnpm add vue3-hash-calendar
```

### 引入

```js
import Vue from 'vue';
import VueHashCalendar from 'vue3-hash-calendar';
import 'vue3-hash-calendar/es/index.css';

Vue.use(VueHashCalendar);
```
