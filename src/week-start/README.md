# week-start

### 介绍

以星期几作为日历每一周的起始星期。可选值：['sunday', 'monday', 'tuesday', 'wednesday', 'thursday', 'friday', 'saturday']

### sunday

```html
<template>
  <vue-hash-calendar week-start="sunday" />
</template>
```

### monday

```html
<template>
  <vue-hash-calendar week-start="monday" />
</template>
```
