# picker-type

### 介绍

选择器类型

single：单选

range：范围选择

multiple：多选

### single

```html
<template>
  <vue-hash-calendar select-type="single" />
</template>
```

### range

```html
<template>
  <vue-hash-calendar select-type="range" />
</template>
```

### multiple

```html
<template>
  <vue-hash-calendar select-type="multiple" />
</template>
```
