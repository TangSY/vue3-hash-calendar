# mark-date

### 介绍

需要被标记的日期，可按不同颜色不同标记类型分组标记（不分组默认蓝色）。标记类型取值：[MarkType](#mark-type)

### 默认

```html
<vue-hash-calendar
  :mark-date="[
      '2022/11/11',
      '2022/12/11',
      '2023/01/11',
      '2023/02/11',
      '2023/03/11',
      '2023/04/11',
      '2023/05/11',
    ]"
/>
```

### 自定义颜色

```html
<vue-hash-calendar
  :mark-date="[
      {
        color: '#f00',
        date: [
          '2022/11/11',
          '2022/12/11',
          '2023/01/11',
          '2023/02/11',
          '2023/03/11',
          '2023/04/11',
          '2023/05/11',
        ],
      },
    ]"
/>
```

### 自定义标记类型

```html
<vue-hash-calendar
  :mark-date="[
      {
        type: 'dot',
        date: [
          '2022/11/11',
          '2022/12/11',
          '2023/01/11',
          '2023/02/11',
          '2023/03/11',
          '2023/04/11',
          '2023/05/11',
        ],
      },
    ]"
/>
```

### 交叉使用

```html
<vue-hash-calendar
  :mark-date="[
      {
        color: '#f00',
        type: 'dot',
        date: ['2022/11/11', '2022/12/11', '2023/04/11'],
      },
      {
        type: 'dot+circle',
        date: ['2023/01/11', '2023/02/11', '2023/03/11'],
      },
      '2023/05/11',
    ]"
/>
```
