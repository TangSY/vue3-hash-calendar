# mark-date

### 介绍

需要被标记的日期，可按不同颜色不同标记类型分组标记（默认蓝色小点）

color: 标记颜色

type: 标记类型，可选：`dot | circle | dot+circle`

date: 需要被标记的日期

### 默认标记方式

```html
<vue-hash-calendar
  :mark-date="[
      `2022/01/01`,
      `2022/01/05`,
      `2022/01/10`,
      `2022/01/15`,
      `2022/01/20`,
      `2022/01/25`,
      `2022/01/28`,
      `2022/01/16`,
      `2022/01/18`,
    ]"
/>
```

### 自定义颜色

```html
<vue-hash-calendar
  :mark-date="[
      {
        color: '#f00',
        date: [
          `2022/01/01`,
          `2022/01/05`,
          `2022/01/10`,
          `2022/01/15`,
          `2022/01/20`,
          `2022/01/25`,
          `2022/01/28`,
          `2022/01/16`,
          `2022/01/18`,
        ],
      },
    ]"
/>
```

### 自定义标记类型

```html
<vue-hash-calendar
  :mark-date="[
      {
        type: 'circle',
        date: [
          `2022/01/01`,
          `2022/01/05`,
          `2022/01/10`,
          `2022/01/15`,
          `2022/01/20`,
          `2022/01/25`,
          `2022/01/28`,
          `2022/01/16`,
          `2022/01/18`,
        ],
      },
    ]"
/>
```

### 交叉使用

```html
<vue-hash-calendar
  :mark-date="[
      {
        color: '#f00',
        type: 'dot+circle',
        date: [
          `2022/01/01`,
          `2022/01/05`,
          `2022/01/10`,
        ],
      },
      {
        color: '#0f0',
        type: 'circle',
        date: [
          `2022/01/15`,
          `2022/01/20`,
          `2022/01/25`,
        ],
      },
      `2022/01/28`,
      `2022/01/16`,
      `2022/01/18`,
    ]"
/>
```
