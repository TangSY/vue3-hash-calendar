# not-current-month-day-class-name

### 介绍

非当前展示月份日期的 className(例如日历前面几天与后面几天灰色部分)。用于修改非当前展示月份日期的默认样式

### 代码演示

```html
<vue-hash-calendar
  not-current-month-day-class-name="not-current-month-day-class-name"
/>
```

```css
.not-current-month-day-class-name {
  color: blue;
  background: red;
}
```
